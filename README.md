## Getting started

If you want to run this project locally, you can:

1. Clone this project

```bash
git clone https://github.com/iscasur/maybelline-catalog.git

```

2. Go to the project's folder

```bash
cd netflix-clone
```

3. Install

```bash
npm install
```

4. Run development enviroment

```bash
npm run dev
```
